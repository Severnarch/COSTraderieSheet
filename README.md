# COS Traderie Sheet
Data retrieval from Traderie for Creatures of Sonaria

## Overview
This project aims to retrieve data from the online gaming trading platform [Traderie](https://traderie.com/), with a primary focus on the section of Traderie dedicated to the [Roblox](https://www.roblox.com) experience [Creatures of Sonaria](https://www.roblox.com/games/5233782396) by [Sonar Studios LLC](https://sonar.games/).
